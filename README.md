# Nextjs Promptmania

A full-stack app including authentication and CRUD functionalities.

## Built with

- NextJS
- TypeScript
- Tailwind
- bcrypt
- mongoDB


## Features:

- Create, update, read, delete prompts
- Everything is saved in database
- Search & filter functionality
- Fully responsive
- Google and credentials auth
